# Office & Location Photos

Place office/location imagery here.

## Recommended files
| File | Usage |
|------|-------|
| `sarajevo-office.jpg` | Primary office photo (contact section, about) |
| `marsala-tita.jpg` | Street/building exterior |
| `workspace.jpg` | Interior / desk environment |
| `hero-bg.jpg` | Optional: full-bleed hero background (if replacing abstract grid) |

## Recommended specs
- Format: JPG or WebP
- Hero/full-bleed: **1920×1080px minimum**, compressed to < 400 KB
- Section images: **1200×800px**, compressed to < 200 KB
- Use [squoosh.app](https://squoosh.app) to convert to WebP and compress
